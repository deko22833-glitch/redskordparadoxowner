# Discord Clone

Полнофункциональный клон Discord с чатом, голосовыми/видео звонками и демонстрацией экрана.

## Функции

- ✅ Регистрация и вход (без email, только username и пароль)
- ✅ Текстовый чат в реальном времени
- ✅ Голосовые звонки
- ✅ Видео звонки
- ✅ Демонстрация экрана (Screen Sharing)
- ✅ Список онлайн пользователей
- ✅ Каналы для общения

## Установка

1. Установите зависимости:
```bash
npm install
```

2. Запустите сервер:
```bash
npm start
```

3. Откройте браузер и перейдите на:
```
http://localhost:3000
```

## Использование

1. Зарегистрируйтесь или войдите в систему
2. Выберите канал для общения
3. Напишите сообщение в чат
4. Нажмите на кнопку звонка для начала голосового/видео звонка
5. Используйте кнопку "Показать экран" для демонстрации экрана

## Технологии

- **Backend**: Node.js, Express, Socket.IO
- **Frontend**: HTML, CSS, JavaScript
- **Database**: SQLite
- **WebRTC**: Для звонков и screen sharing
- **Authentication**: JWT

## Структура проекта

```
.
├── server.js          # Сервер Express + Socket.IO
├── database.js        # Работа с базой данных SQLite
├── index.html         # Главная страница приложения
├── login.html         # Страница входа/регистрации
├── script.js          # Клиентская логика (чат, звонки)
├── auth.js            # Логика аутентификации
├── styles.css         # Стили приложения
└── package.json       # Зависимости проекта
```

## Примечания

- Для работы звонков требуется HTTPS в продакшене (или localhost)
- WebRTC использует STUN серверы Google для NAT traversal
- База данных SQLite создается автоматически при первом запуске
