# Инструкция по установке и запуску

## Быстрый старт

1. **Установите Node.js** (если еще не установлен)
   - Скачайте с https://nodejs.org/
   - Рекомендуемая версия: 18.x или выше

2. **Установите зависимости**
   ```bash
   npm install
   ```

3. **Запустите сервер**
   ```bash
   npm start
   ```

4. **Откройте браузер**
   - Перейдите на http://localhost:3000
   - Зарегистрируйтесь (только username и пароль, без email)
   - Начните общаться!

## Функции

### Чат
- Напишите сообщение в поле ввода и нажмите Enter
- Сообщения отображаются в реальном времени
- Переключайтесь между каналами

### Звонки
- **Голосовой звонок**: Нажмите кнопку с телефоном
- **Видео звонок**: Нажмите кнопку с камерой
- **Показать экран**: Нажмите кнопку с экраном (работает во время звонка)

### Управление звонком
- **Отключить микрофон**: Кнопка микрофона
- **Отключить камеру**: Кнопка камеры
- **Завершить звонок**: Кнопка с трубкой или крестик

## Структура файлов

Все файлы находятся в одной папке:
- `server.js` - сервер Node.js
- `database.js` - работа с базой данных
- `index.html` - главная страница
- `login.html` - страница входа/регистрации
- `script.js` - логика чата и звонков
- `auth.js` - логика аутентификации
- `styles.css` - стили главной страницы
- `auth.css` - стили страницы входа
- `package.json` - зависимости проекта

## Регистрация

Регистрация работает **без email**:
- Username (минимум 3 символа)
- Password (минимум 6 символов)
- Подтверждение пароля (только при регистрации)

## Примечания

- База данных SQLite создается автоматически при первом запуске
- Для работы звонков в браузере требуется разрешение на использование камеры и микрофона
- Screen sharing работает только во время активного звонка
- Все данные хранятся локально в файле `discord_clone.db`
